# Changelog

## 0.2.0 - UI Redesign "Quiet Health"

### Changed

- 全站 UI 重构为 Apple 风格设计系统,业务逻辑未变。
- styles.css 完全重写:token 驱动(颜色/圆角/阴影全部来自变量),0 个 `!important`,字重只保留 400/500/600/700。
- 深浅色主题改为 token 层切换,组件不再单独适配深色。
- 导航改为底部悬浮固定的 Liquid Glass Tab Bar,移动端和桌面端统一;玻璃材质仅用于悬浮层(Tab Bar、Toast、弹层背板)。
- 头部改为 iOS 大标题样式:日期大标题 + 日期选择/新增/主题/更多四个图标按钮。
- 总览页信息架构简化:今日摄入卡(大数字 + 余量 + 照片)→ 四餐分组列表 → 常吃食物;右栏为小日历、目标环、营养比例、7 日复盘。
- 减脂复盘与"做得好/需要注意"洞察卡移至趋势页。
- 顶栏工具按钮(同步/导入/导出/手机查看/快捷指令/退出)收纳进"更多操作"面板。
- 弹层在移动端呈现为底部 Sheet。
- 移除旧的桌面侧边栏、玻璃拟态卡片和多层日期覆盖样式。

## 0.1.0 - Initial Cloud Version

### Added

- 云端饮食记录应用基础版本。
- Supabase Email Magic Link 登录。
- Supabase Postgres 数据表和 Storage 照片存储。
- 早餐、午餐、晚餐、其他餐次记录。
- 食物明细：重量、热量、蛋白质、碳水、脂肪、膳食纤维、备注和照片。
- 手动新增、编辑、删除饮食记录。
- Gemini 餐食照片分析入口。
- 体重、BMI、体脂、肌肉、水分、基础代谢、内脏脂肪等身体数据记录。
- Gemini 体脂秤截图识别入口。
- 身体数据趋势和身体分布图。
- Apple 健康导出 zip 解析和导入。
- iPhone Shortcuts 运动数据同步 API。
- 运动总览、活动圆环、趋势和健康分类展示。
- 照片库按日期展示餐食图片。
- 常吃食物快捷添加模块。
- 目标系统和减脂复盘建议。
- 响应式桌面端和移动端布局。
- 浅色/深色模式切换。

### Changed

- 从本地静态饮食日记升级为 Vercel + Supabase 云端架构。
- 将照片从本地文件引用迁移为 Supabase Storage 路径。
- 移动端布局改为顶部固定导航和紧凑卡片流。
- 视觉风格向 Apple Health / Apple Fitness 的低饱和、轻量卡片系统靠拢。

### Security

- 真实环境变量不提交，仅保留 `.env.example`。
- service role key 仅用于服务端函数和本地脚本。
- `.gitignore` 排除构建产物、缓存、日志、Vercel 状态和本地环境文件。
